# PPT动画与切换效果

## 本章概览

动画和切换效果能让PPT更生动,增强表达效果。但过度使用会适得其反。本章介绍如何恰当使用动画提升演示效果。

**学习目标**
- 掌握幻灯片切换效果
- 学会设置对象动画
- 理解动画时序控制
- 避免动画使用误区

---

## 34.1 幻灯片切换

### 34.1.1 切换效果

**应用切换**
```
切换选项卡 → 选择效果:

平滑类:
- 淡出: 渐变过渡
- 推进: 推入效果
- 擦除: 擦除显示

动态类:
- 翻转: 翻页效果
- 立方体: 3D旋转
- 缩放: 放大缩小

应用到全部: 所有幻灯片统一切换
```

**推荐切换**
```
商务汇报: 淡出、推进
产品发布: 平滑、变形
教学培训: 擦除、淡出

避免: 过于花哨的3D效果
```

### 34.1.2 切换设置

**持续时间**
```
切换 → 持续时间
推荐: 0.5-1.0秒
过快: 看不清
过慢: 拖沓
```

**换片方式**
```
单击鼠标时: 手动控制(推荐)
设置自动换片时间: 自动播放
勾选"单击鼠标时"同时设置时间: 两者皆可
```

### 34.1.3 平滑切换(重点)

**什么是平滑**
```
PowerPoint自动识别两页间相同对象
创建平滑过渡动画

适用版本: Office 2016及以上
```

**使用方法**
```
1. 复制幻灯片(Ctrl + D)
2. 调整第二页对象位置/大小/颜色
3. 第二页切换效果选择"平滑"

效果: 对象自动移动/缩放/变色
```

**应用场景**
```
- Logo移动
- 图表数据变化
- 图片缩放位移
- 文字渐变色彩
```

## 34.2 动画效果

### 34.2.1 动画类型

**进入动画**
```
对象出现的方式

常用:
- 淡出: 渐现
- 飞入: 从外飞入
- 浮入: 浮现
- 擦除: 擦出

动画 → 添加动画 → 进入
```

**强调动画**
```
已存在对象的强调效果

常用:
- 放大/缩小
- 颜色变化
- 旋转
- 脉冲

用途: 引起注意
```

**退出动画**
```
对象消失的方式

常用:
- 淡出: 渐隐
- 飞出: 飞走
- 消失: 直接消失

用途: 切换内容
```

**路径动画**
```
自定义移动路径

动画 → 添加动画 → 更多路径
选择路径类型或自定义

应用: 复杂移动轨迹
```

### 34.2.2 设置动画

**添加动画**
```
1. 选中对象
2. 动画 → 添加动画
3. 选择效果

同一对象可添加多个动画
```

**动画选项**
```
动画 → 效果选项

方向: 从上/从下/从左/从右
形状: 圆形/菱形/十字
声音: 添加音效(少用)
```

**动画窗格**
```
动画 → 动画窗格

显示:
- 所有动画列表
- 播放顺序
- 触发方式
- 持续时间

操作:
- 调整顺序(拖动)
- 删除动画
- 复制动画
```

### 34.2.3 动画计时

**开始方式**
```
单击时: 点击触发(默认)
与上一动画同时: 同时播放
在上一动画之后: 自动接续
```

**持续时间**
```
推荐: 0.3-0.5秒(快速)
文字: 0.5-1.0秒(适中)
图片: 1.0-2.0秒(慢速)
```

**延迟**
```
动画开始前的等待时间
用途: 控制节奏
```

## 34.3 动画技巧

### 34.3.1 文字动画

**整体动画**
```
选中文本框 → 添加动画
整个文本框一起动
```

**逐行/逐字动画**
```
效果选项 → 文本动画:
- 整批发送: 一次全部
- 按字/词: 逐字出现
- 按段落: 逐段出现

延迟: 设置字词间隔
```

**使用场景**
```
标题: 整体淡出
列表: 逐行飞入
强调: 逐字打字效果

避免: 大段文字逐字
```

### 34.3.2 组合动画

**序列动画**
```
多个对象按顺序播放

设置:
1. 为每个对象添加动画
2. 调整"开始"为"在上一动画之后"
3. 动画窗格拖动排序

应用: 流程图步骤展示
```

**同时动画**
```
多个对象同时播放

设置:
开始 → 与上一动画同时

应用: 多元素统一出现
```

### 34.3.3 触发器

**什么是触发器**
```
点击特定对象时触发动画
实现交互效果
```

**设置触发器**
```
1. 选中要播放动画的对象A
2. 添加动画
3. 动画窗格 → 动画右侧下拉 → 计时
4. 触发器 → 单击下列对象时启动效果
5. 选择触发对象B

效果: 点击B时,A播放动画
```

**应用场景**
```
- 点击按钮显示答案
- 点击图标展开说明
- 交互式菜单
```

## 34.4 动画最佳实践

### 34.4.1 使用原则

**少即是多**
```
不是每个元素都需要动画
重点内容才添加动画
全篇动画不超过30%
```

**一致性**
```
相同类型元素使用相同动画
进入动画统一(如都用淡出)
方向统一(如都从左飞入)
```

**速度适中**
```
过快: 看不清
过慢: 拖沓
推荐: 0.3-0.5秒

总时长: 不超过2秒
```

### 34.4.2 场景应用

**商务汇报**
```
切换: 淡出/推进
动画: 淡出/擦除
避免: 旋转/飞入等夸张效果

原则: 专业、低调
```

**产品发布**
```
切换: 平滑
动画: 放大/缩小/路径
可适度炫酷

原则: 吸引眼球
```

**教学培训**
```
切换: 淡出
动画: 逐行出现
强调: 颜色脉冲

原则: 清晰、有节奏
```

### 34.4.3 避免错误

**常见误区**
```
❌ 所有元素都加动画
❌ 使用过多不同动画
❌ 动画时间过长
❌ 声音效果
❌ 过度旋转翻转

✓ 只为重点添加
✓ 统一动画风格
✓ 快速简洁
✓ 静音
✓ 简单直接
```

## 34.5 高级动画

### 34.5.1 擦除动画

**方向控制**
```
效果选项 → 方向
从上/下/左/右

技巧:
文字从左擦除: 阅读习惯
图表从下擦除: 增长感
```

### 34.5.2 路径动画

**预设路径**
```
添加动画 → 更多动画 → 动作路径
选择路径: 直线/弧形/圆形等
```

**自定义路径**
```
添加动画 → 动作路径 → 自定义路径
鼠标绘制移动轨迹

应用: 产品移动演示
```

**路径编辑**
```
右键路径 → 编辑顶点
调整路径形状
```

### 34.5.3 动画组合

**淡入+移动**
```
1. 添加淡出(进入)
2. 添加自定义路径(动作路径)
3. 两个动画都设为"与上一动画同时"

效果: 边淡入边移动
```

**放大+旋转**
```
1. 添加放大(强调)
2. 添加旋转(强调)
3. 设为同时播放

效果: 边放大边旋转
```

## 34.6 实战案例

### 案例1: 流程图动画

**需求**
步骤1 → 步骤2 → 步骤3 依次出现

**设置**
```
1. 步骤1: 淡出,单击时
2. 箭头1: 擦除,在上一动画之后
3. 步骤2: 淡出,在上一动画之后
4. 箭头2: 擦除,在上一动画之后
5. 步骤3: 淡出,在上一动画之后

持续时间: 均为0.5秒
```

### 案例2: 数据图表动画

**需求**
图表按系列逐条显示

**设置**
```
1. 选中图表
2. 添加动画 → 擦除
3. 效果选项 → 按系列

效果: 柱子/线条逐个出现
```

### 案例3: 列表逐行显示

**需求**
要点逐条出现,已显示的变灰

**设置**
```
1. 文本框添加动画 → 淡出
2. 效果选项 → 按段落
3. 右键动画 → 效果 → 效果选项卡
4. 动画播放后: 设置颜色(浅灰)

效果: 新出现的亮色,旧的变灰
```

## 34.7 演示技巧

### 34.7.1 排练计时

**排练功能**
```
幻灯片放映 → 排练计时

开始演示:
- 按节奏讲解
- 切换时录制时间
- 结束后保存计时

效果: 自动播放时使用录制的时间
```

**用途**
```
- 视频录制
- 无人演示
- 展会循环播放
```

### 34.7.2 放映控制

**演示快捷键**
```
F5: 从头播放
Shift + F5: 从当前播放
Esc: 结束放映

B: 黑屏
W: 白屏
数字+Enter: 跳转到指定页

Ctrl + P: 显示笔
Ctrl + E: 清除标注
```

### 34.7.3 演讲者视图

**启用演讲者视图**
```
幻灯片放映 → 勾选"使用演讲者视图"

显示内容:
- 当前幻灯片
- 下一张预览
- 备注
- 计时器

优势: 观众看内容,演讲者看提示
```

## 本章小结

**核心要点**
1. **切换**: 淡出/推进/平滑,简洁专业
2. **动画**: 进入/强调/退出/路径
3. **计时**: 开始方式、持续时间、延迟
4. **原则**: 少用、统一、快速

**推荐组合**
```
商务: 淡出切换 + 淡出/擦除动画
发布: 平滑切换 + 放大/路径动画
教学: 淡出切换 + 逐行/强调动画
```

**避免**
- 过度动画
- 声音效果
- 长时间动画
- 不统一风格
