# PPT多媒体应用

## 本章概览

多媒体元素(音频、视频)能让PPT更生动,但使用不当会分散注意力。本章介绍如何恰当使用多媒体提升演示效果。

**学习目标**
- 掌握音频插入与设置
- 学会视频嵌入与播放
- 了解屏幕录制功能
- 掌握超链接与交互

---

## 36.1 音频应用

### 36.1.1 插入音频

**方法**
```
插入 → 音频:
- 此设备上的音频: 本地文件
- 录制音频: 现场录制
- 联机音频: 在线搜索(已移除)

支持格式: MP3, WAV, M4A
```

**音频图标**
```
插入后显示音频图标
可拖动调整位置
建议放在不显眼位置或隐藏
```

### 36.1.2 音频设置

**播放选项**
```
音频工具 → 播放:

开始:
- 单击时: 手动播放
- 自动: 自动播放
- 跨幻灯片播放: 背景音乐

音量: 低/中/高/静音
隐藏: 放映时隐藏图标
循环播放: 重复播放
```

**应用场景**
```
背景音乐: 跨幻灯片播放+循环+隐藏
音效: 单击播放+音量中
旁白: 自动播放
```

### 36.1.3 录制音频

**录制旁白**
```
插入 → 音频 → 录制音频
点击录制按钮
讲解内容
停止录制
命名并确定

用途: 课程讲解、产品介绍
```

## 36.2 视频应用

### 36.2.1 插入视频

**方法**
```
插入 → 视频:
- 此设备: 本地视频文件
- 联机视频: YouTube/优酷链接
- 屏幕录制: 录制屏幕

支持格式: MP4(推荐), WMV, AVI
编码: H.264
```

**嵌入 vs 链接**
```
嵌入: 视频内置到PPT
- 优点: 便携,不丢失
- 缺点: 文件变大

链接: 引用外部视频
- 优点: 文件小
- 缺点: 移动后可能找不到

推荐: 嵌入(确保文件≤100MB)
```

### 36.2.2 视频设置

**播放选项**
```
视频工具 → 播放:

开始: 单击时/自动/跨幻灯片播放
音量: 调节音量
循环播放: 重复播放
全屏播放: 占满屏幕
播放完后倒回: 回到首帧
```

**视频样式**
```
视频工具 → 格式:
- 视频样式: 边框阴影
- 视频形状: 裁剪为圆形等
- 视频效果: 映像、发光
```

**裁剪视频**
```
视频工具 → 播放 → 剪裁视频
设置开始时间和结束时间
只播放指定片段

用途: 只播放关键部分
```

### 36.2.3 视频封面

**设置封面**
```
方法1: 海报框架
- 播放到想要的画面
- 视频工具 → 格式 → 海报框架 → 当前框架

方法2: 自定义图片
- 右键视频 → 设置视频格式
- 海报框架 → 文件中的图片
```

## 36.3 屏幕录制

### 36.3.1 录制屏幕

**启动录制**
```
插入 → 屏幕录制
选择录制区域
点击录制按钮(或Win+Shift+R)
执行操作
Win+Shift+Q停止录制

自动插入PPT
```

**应用场景**
```
- 软件操作演示
- 网页浏览演示
- 应用使用教程
```

### 36.3.2 录制技巧

**准备工作**
```
- 关闭不相关窗口
- 准备操作流程
- 调整窗口大小适中
- 确保录制区域清晰
```

**录制中**
```
- 操作放慢节奏
- 鼠标移动清晰
- 避免误操作
```

## 36.4 超链接与交互

### 36.4.1 超链接

**创建超链接**
```
选中对象(文字/图片/形状)
插入 → 超链接(或Ctrl+K)

链接到:
- 网页: 输入URL
- 本文档中的位置: 跳转到指定幻灯片
- 新建文档: 创建新文件
- 电子邮件地址: 发送邮件
```

**应用场景**
```
目录页: 点击章节跳转
参考资料: 链接到网页
返回按钮: 返回目录
```

### 36.4.2 动作按钮

**插入动作按钮**
```
插入 → 形状 → 动作按钮

类型:
- 后退/前进: 上下翻页
- 开始/结束: 跳转首尾页
- 返回: 返回上一页
- 文档: 打开文件
- 声音: 播放音频
```

**设置动作**
```
插入后自动弹出"动作设置"
或右键按钮 → 编辑超链接

单击鼠标/鼠标移过
超链接到: 选择目标
```

### 36.4.3 交互式目录

**创建目录**
```
1. 创建目录页
2. 列出各章节标题
3. 为每个标题添加超链接到对应幻灯片
4. 各章节首页添加"返回目录"按钮

效果: 可跳转的交互目录
```

## 36.5 导出与录制

### 36.5.1 导出视频

**导出为视频文件**
```
文件 → 导出 → 创建视频

设置:
- 质量: 超高清/高清/标清
- 是否使用录制的计时和旁白
- 每张幻灯片秒数(如5秒)

保存为MP4

用途: 上传视频平台、无人演示
```

### 36.5.2 录制幻灯片演示

**录制演示**
```
幻灯片放映 → 录制幻灯片演示

录制:
- 旁白(麦克风)
- 时间计时
- 墨迹和激光笔

保存: 嵌入PPT

播放: 自动包含旁白和计时
```

**应用**
```
在线课程: 录制讲解
培训视频: 录制演示
```

## 36.6 最佳实践

### 36.6.1 音视频使用原则

**必要性**
```
✓ 产品演示视频
✓ 操作教程
✓ 客户证言视频
✓ 数据可视化动画

✗ 无关的装饰性视频
✗ 过长的视频(>2分钟)
✗ 背景音乐干扰讲解
```

**技术要求**
```
视频:
- 格式: MP4(H.264)
- 分辨率: ≥1080p
- 大小: ≤100MB(嵌入)
- 时长: ≤2分钟

音频:
- 格式: MP3
- 音质: 清晰无杂音
- 音量: 适中统一
```

### 36.6.2 避免问题

**常见问题**
```
❌ 视频无法播放
原因: 编码不支持
解决: 转换为MP4(H.264)

❌ 文件过大
原因: 视频未压缩
解决: 压缩视频或外部链接

❌ 音频丢失
原因: 链接文件未携带
解决: 嵌入音频到PPT
```

### 36.6.3 文件管理

**组织文件**
```
项目文件夹/
├── PPT文件.pptx
├── 视频/
│   ├── 产品演示.mp4
│   └── 操作教程.mp4
├── 音频/
│   └── 背景音乐.mp3
└── 图片/
    └── ...

打包: 将整个文件夹打包发送
```

## 36.7 实战案例

### 案例1: 产品演示PPT

**结构**
```
1. 封面: 产品名称
2. 目录: 交互式跳转
3. 功能介绍: 文字+图片
4. 实际演示: 嵌入演示视频(30秒)
5. 客户评价: 视频证言(20秒)
6. 联系方式
```

**多媒体使用**
```
- 演示视频: 自动播放,静音,循环
- 证言视频: 单击播放,有声
- 背景音乐: 跨幻灯片,低音量(可选)
```

### 案例2: 在线课程PPT

**结构**
```
1. 课程大纲
2. 知识点讲解: 屏幕录制操作
3. 练习题: 交互按钮显示答案
4. 总结
```

**多媒体使用**
```
- 录制演示: 包含旁白
- 导出视频: 上传学习平台
```

## 本章小结

**核心要点**
1. **音频**: 插入、播放设置、录制旁白
2. **视频**: 嵌入、播放控制、裁剪封面
3. **录制**: 屏幕录制、幻灯片演示录制
4. **交互**: 超链接、动作按钮、交互目录
5. **导出**: 导出为视频文件

**使用建议**
- 多媒体为内容服务,不喧宾夺主
- 视频时长控制在2分钟内
- 确保音视频质量
- 嵌入重要多媒体文件
- 测试所有链接和播放
