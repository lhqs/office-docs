# PPT协作与分享

## 本章概览

现代工作中,PPT常需要多人协作完成。本章介绍如何高效协作、审阅修订、安全分享演示文稿。

**学习目标**
- 掌握多人协作方法
- 学会审阅和修订
- 了解文件保护技巧
- 熟悉云端协作工具

---

## 39.1 协作编辑

### 39.1.1 共同创作

**OneDrive/SharePoint协作**
```
前提: 文件保存在云端

步骤:
1. 文件 → 共享
2. 邀请人员(输入邮箱)
3. 设置权限: 可编辑/可查看
4. 发送邀请

实时协作:
- 查看其他人光标
- 同时编辑不同页
- 自动保存合并
```

**Office 365网页版**
```
访问: office.com
登录账号
打开PPT文件

优势:
- 浏览器直接编辑
- 无需安装软件
- 跨平台协作
- 版本历史记录

限制:
- 部分高级功能缺失
- 需要网络连接
```

### 39.1.2 协作流程

**分工协作**
```
方式1: 按幻灯片分工
- A负责: 第1-5页
- B负责: 第6-10页
- C负责: 第11-15页
最后合并检查

方式2: 按职能分工
- 内容: 撰写文字内容
- 设计: 美化排版设计
- 数据: 图表数据分析
协同完成

方式3: 轮流修改
- 版本1: A创建初稿
- 版本2: B修改完善
- 版本3: C最终审定
按顺序传递
```

**版本控制**
```
命名规范:
项目名_版本_日期_作者.pptx
例: 季度汇报_v1.0_20240101_张三.pptx
    季度汇报_v1.1_20240102_李四.pptx
    季度汇报_Final_20240103_王五.pptx

版本说明:
v0.x: 草稿阶段
v1.x: 初稿阶段
v2.x: 修改阶段
Final: 最终版本
```

## 39.2 审阅与修订

### 39.2.1 批注功能

**添加批注**
```
审阅 → 新建批注
或右键 → 新建批注
或Ctrl + Alt + M

批注内容:
- 修改建议
- 问题疑问
- 补充说明

批注显示:
侧边栏显示所有批注
点击跳转到对应位置
```

**回复批注**
```
点击批注 → 回复
可以对话式讨论

批注管理:
- 已解决: 标记完成
- 删除批注
- 显示/隐藏批注
```

### 39.2.2 修订追踪

**比较合并**
```
审阅 → 比较

选择两个版本:
- 原始文件
- 修订文件

查看差异:
- 删除内容: 红色
- 新增内容: 绿色
- 修改内容: 蓝色

接受/拒绝更改:
逐项审查决定是否采纳
```

### 39.2.3 审阅流程

**标准审阅流程**
```
1. 创建初稿
   ↓
2. 共享给审阅者
   ↓
3. 审阅者添加批注
   ↓
4. 创建者查看批注
   ↓
5. 修改并标记完成
   ↓
6. 删除批注,定稿

技巧:
- 批注用不同颜色区分审阅者
- 重要批注用红色标记
- 定期整理已解决批注
```

## 39.3 文件保护

### 39.3.1 密码保护

**设置密码**
```
文件 → 信息 → 保护演示文稿 → 用密码进行加密

输入密码(建议强密码)
确认密码
保存文件

打开文件需要输入密码
```

**限制编辑**
```
文件 → 信息 → 保护演示文稿 → 限制编辑

设置权限:
- 允许查看但不能编辑
- 允许打印但不能修改
- 允许批注但不能更改内容
```

### 39.3.2 标记为最终状态

**设置最终状态**
```
文件 → 信息 → 保护演示文稿 → 标记为最终状态

效果:
- 文件只读
- 顶部黄色提示条
- 可解除(点击仍然编辑)

用途:
- 发布最终版本
- 防止误修改
- 提示文件已完成
```

### 39.3.3 删除敏感信息

**检查文档**
```
文件 → 信息 → 检查问题 → 检查文档

检查项目:
- 批注和修订
- 文档属性(作者等)
- 隐藏幻灯片
- 隐藏文字
- 演示文稿备注

删除敏感信息:
勾选要删除的项目
全部删除
```

## 39.4 导出与分享

### 39.4.1 导出格式

**PDF格式**
```
文件 → 导出 → 创建PDF/XPS文档

优点:
- 跨平台查看
- 不可编辑(保护内容)
- 文件较小
- 保留格式

设置:
- 质量: 标准/最小
- 包含批注: 可选
- 范围: 全部/部分页
```

**视频格式**
```
文件 → 导出 → 创建视频

设置:
- 质量: 超高清(1080p)/高清(720p)/标清(480p)
- 使用录制的计时和旁白: 是/否
- 每张幻灯片秒数: 5秒(默认)

保存为: .mp4文件

用途:
- 上传视频平台
- 无人值守播放
- 学习资料分享
```

**图片格式**
```
文件 → 导出 → 更改文件类型 → PNG/JPEG

每张幻灯片导出为一张图片
用途:
- 社交媒体分享
- 网页展示
- 邮件附件(图片版)
```

### 39.4.2 压缩文件

**压缩图片**
```
文件 → 信息 → 压缩图片

选项:
- 应用于: 仅此图片/所有图片
- 目标输出:
  - 电子邮件(96ppi)
  - Web(150ppi)
  - 打印(220ppi)
  - 使用默认分辨率

效果:
大幅减小文件大小
```

**压缩媒体**
```
文件 → 信息 → 压缩媒体

质量选项:
- 演示品质: 保持高质量
- Internet质量: 适度压缩
- 低质量: 最大压缩

适用: 嵌入的视频文件
```

### 39.4.3 分享方式

**邮件分享**
```
文件 → 共享 → 电子邮件

选项:
- 作为附件发送
- 作为PDF发送
- 作为XPS发送
- 发送链接(OneDrive)

建议:
- 附件<10MB
- 大文件用云盘链接
- 注明版本和用途
```

**云盘分享**
```
OneDrive/Google Drive:
1. 上传文件到云盘
2. 获取分享链接
3. 设置权限(查看/编辑)
4. 复制链接发送

优势:
- 不受文件大小限制
- 实时更新
- 权限可控
```

**演示文稿打包**
```
文件 → 导出 → 将演示文稿打包成CD

包含:
- PPT文件
- 嵌入字体
- 链接文件
- PowerPoint Viewer(可选)

用途:
- 在无PPT的电脑演示
- 打包所有相关文件
```

## 39.5 在线演示

### 39.5.1 Office演示服务

**在线演示**
```
幻灯片放映 → 演示幻灯片 → 联机演示

步骤:
1. 启用演示服务
2. 获取分享链接
3. 发送给观众
4. 开始演示

观众查看:
- 浏览器打开链接
- 实时查看演示
- 无需安装软件
- 支持多人同时观看
```

### 39.5.2 Teams/Zoom共享

**会议中共享PPT**
```
Teams/Zoom:
1. 加入会议
2. 共享屏幕
3. 选择PowerPoint窗口
4. 开始演示

PowerPoint Live(Teams):
- 观众端独立查看进度
- 可以自己翻页
- 演讲者控制主进度
```

### 39.5.3 录制演示

**录制幻灯片演示**
```
幻灯片放映 → 录制幻灯片演示

录制内容:
- 幻灯片切换时间
- 旁白(麦克风)
- 墨迹和激光笔

保存:
嵌入到演示文稿
或导出为视频

用途:
- 异步培训
- 自学课程
- 产品演示视频
```

## 39.6 协作工具对比

### 39.6.1 工具对比

**主流协作工具**
```
Microsoft PowerPoint + OneDrive:
✓ 强大功能
✓ 桌面和网页版
✓ Office 365生态
✗ 需要订阅(部分功能)

Google Slides:
✓ 完全免费
✓ 浏览器访问
✓ 实时协作流畅
✗ 功能相对简单

WPS演示 + 金山文档:
✓ 国内访问快
✓ 免费基础功能
✓ 模板丰富
✗ 高级功能收费

Prezi:
✓ 创新展示方式
✓ 云端协作
✗ 学习曲线陡
✗ 收费
```

### 39.6.2 选择建议

**根据场景选择**
```
企业内部协作:
→ Office 365 + SharePoint/Teams

跨组织协作:
→ Google Slides(兼容性好)

个人使用:
→ PowerPoint桌面版 + OneDrive

快速分享:
→ 导出PDF/视频
```

## 39.7 实战案例

### 案例1: 跨部门协作汇报

**场景**
```
需求: 3个部门共同完成季度汇报
人员: 市场部、销售部、产品部各1人
时间: 3天完成
```

**协作流程**
```
Day 1:
- 产品部建立框架(目录、版式)
- 保存到SharePoint
- 共享给其他部门

Day 2:
- 各部门并行编辑各自章节
- 市场部: 第2-4页
- 销售部: 第5-7页
- 产品部: 第8-10页
- 实时同步

Day 3:
- 产品部统一美化
- 各部门审阅批注
- 修改定稿
```

### 案例2: 外部客户演示

**场景**
```
需求: 给客户远程演示方案
方式: 线上会议
```

**准备清单**
```
技术准备:
□ 测试Teams/Zoom共享屏幕
□ 关闭通知避免打扰
□ 准备备用设备

文件准备:
□ 保存最新版本
□ 导出PDF备份
□ 检查删除敏感信息

演示技巧:
□ 使用演讲者视图
□ 准备备注
□ 测试翻页流畅度
```

## 本章小结

**核心要点**
1. **协作编辑**: OneDrive共享+实时编辑
