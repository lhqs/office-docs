# Excel界面与数据输入规范

## 引言

Excel是数据处理的核心工具,掌握Excel从规范的数据输入开始。本章将系统讲解Excel界面布局和专业的数据输入方法。

## 一、Excel界面全解析

### 1.1 工作簿与工作表

**工作簿(Workbook):**
- 扩展名:.xlsx/.xls/.xlsm
- 一个文件就是一个工作簿
- 可包含多个工作表

**工作表(Worksheet):**
- 默认名称:Sheet1, Sheet2...
- 单个工作表:1,048,576行 × 16,384列
- 列标:A-XFD
- 行号:1-1,048,576

**工作表操作:**
```
新建:Shift+F11
重命名:双击工作表标签
复制:Ctrl+拖动工作表标签
移动:直接拖动工作表标签
删除:右键→删除
隐藏:右键→隐藏
保护:右键→保护工作表
```

## 二、单元格基础

### 2.1 单元格引用

**基本引用:**
- A1:第1行第1列
- B10:第10行第2列

**相对引用:**
```
公式:=A1+B1
复制到下一行:=A2+B2(自动调整)
```

**绝对引用:**
```
公式:=$A$1+B1
复制到任何位置:=$A$1+B2($A$1不变)

快捷键:F4(切换引用类型)
- A1:相对引用
- $A$1:绝对引用
- $A1:列绝对,行相对
- A$1:列相对,行绝对
```

**混合引用:**
```
应用场景:九九乘法表
=B$1*$A2
- B$1:列相对,行绝对(B列固定在第1行)
- $A2:列绝对,行相对(A列,行号变化)
```

### 2.2 选择技巧

**快速选择:**
- Ctrl+A:全选工作表
- Ctrl+Shift+End:选到最后一个有数据的单元格
- Ctrl+Shift+方向键:选择到数据区域边缘
- Shift+空格:选择整行
- Ctrl+空格:选择整列

**名称框选择:**
```
在名称框输入:
- A1:B10 → 选择区域
- A:A → 选择整列
- 1:1 → 选择整行
- A1,C3,E5 → 选择多个不连续单元格
```

**定位功能(Ctrl+G或F5):**
- 空值
- 公式
- 批注
- 可见单元格
- 条件格式
- 数据有效性

## 三、数据输入规范

### 3.1 数据类型

**1. 文本**
```
输入方法:
- 直接输入
- 数字前加单引号:'001(显示001而非1)
- 以等号开头要显示文本:加单引号'=1+1

特点:
- 左对齐
- 无法参与数值计算
```

**2. 数值**
```
输入方法:
- 直接输入数字:123
- 千分位:输入1,234或设置格式
- 负数:输入-100或(100)

特点:
- 右对齐
- 可以参与计算
- 显示格式可变(常规、货币、百分比等)
```

**3. 日期和时间**
```
输入方法:
- 日期:2025-12-21、2025/12/21、2025.12.21
- 时间:10:30、10:30:45
- 日期时间:2025-12-21 10:30

快捷键:
- Ctrl+;:插入当前日期
- Ctrl+Shift+;:插入当前时间

本质:
- 日期是数字(从1900-1-1开始的天数)
- 2025-12-21 = 45985天
```

**4. 公式**
```
输入方法:
- 以=开头:=A1+B1
- 以+或-开头:+A1+B1

特点:
- 单元格显示计算结果
- 编辑栏显示公式
- 自动重算
```

### 3.2 数据验证(数据有效性)

**作用:**
- 限制输入内容
- 提供下拉选项
- 防止错误数据

**设置方法:**
```
数据→数据验证

验证条件:
1. 整数:限制输入整数范围
2. 小数:限制小数范围
3. 列表:下拉选项
4. 日期:日期范围
5. 时间:时间范围
6. 文本长度:字符数限制
7. 自定义:公式判断
```

**案例:性别下拉选择**
```
1. 选中单元格区域
2. 数据→数据验证
3. 允许:列表
4. 来源:男,女(逗号分隔)
5. 确定
```

**案例:年龄范围限制**
```
1. 数据→数据验证
2. 允许:整数
3. 数据:介于
4. 最小值:0
5. 最大值:120
6. 输入信息:请输入0-120之间的年龄
7. 出错警告:年龄必须在0-120之间
```

### 3.3 自动填充

**填充柄:**
- 鼠标悬停在单元格右下角
- 出现黑色十字光标
- 拖动填充

**智能填充:**
```
1. 数字序列
   输入:1,2 → 选中 → 拖动 → 3,4,5...

2. 日期序列
   输入:周一 → 拖动 → 周二,周三...
   输入:1月 → 拖动 → 2月,3月...

3. 自定义序列
   文件→选项→高级→编辑自定义列表
   添加:第一季度,第二季度,第三季度,第四季度
```

**填充选项:**
- 复制单元格
- 填充序列
- 仅填充格式
- 不带格式填充
- 填充工作日
- 填充月份
- 填充年份

**快捷填充(Ctrl+E,Excel 2013+):**
```
案例:拆分姓名
A列:张三
B列手动输入:张
选中B列,Ctrl+E → 自动识别模式,填充其他姓

应用:
- 文本合并
- 文本拆分
- 格式转换
- 数据清洗
```

### 3.4 批量输入技巧

**同时输入多个单元格:**
```
1. 选中多个单元格(Ctrl+点击或拖选)
2. 输入内容
3. Ctrl+Enter → 所有选中单元格填入相同内容
```

**自动完成:**
```
在列中输入重复内容时:
- 输入首字母,Alt+↓:下拉已输入过的内容
- 输入前几个字符:自动匹配并提示
```

**选择性粘贴:**
```
Ctrl+Alt+V或Alt+E+S

选项:
- 全部
- 公式
- 数值:仅粘贴计算结果
- 格式
- 批注
- 有效性验证
- 列宽
- 公式和数字格式
- 数值和数字格式

运算:
- 无
- 加:原值+复制值
- 减
- 乘
- 除

其他:
- 跳过空单元格
- 转置:行列互换
```

## 四、数据输入最佳实践

### 4.1 表格设计原则

**原则1:一行一记录**
```
正确:
姓名 | 年龄 | 性别
张三 | 25  | 男
李四 | 30  | 女

错误:
姓名 | 张三 | 李四
年龄 | 25  | 30
```

**原则2:一列一字段**
```
正确:
姓  | 名  | 年龄
张  | 三  | 25

错误:
姓名    | 年龄
张三    | 25
```

**原则3:标题行唯一**
```
正确:
姓名 | 年龄 | 性别
(数据从第2行开始)

错误:
员工信息表
姓名 | 年龄 | 性别
(多行标题)
```

**原则4:避免合并单元格**
```
合并单元格的问题:
- 无法排序
- 筛选异常
- 公式引用错误
- 数据透视表无法识别

替代方案:
- 使用居中对齐
- 使用辅助列
```

**原则5:统一数据格式**
```
日期统一:
✓ 2025-12-21
✗ 2025/12/21、25.12.21(混用)

数字统一:
✓ 1000、2000
✗ 1000、2千(混用)

单位统一:
✓ 单独列标注单位
✗ 单元格内含单位(如:100元)
```

### 4.2 命名规范

**工作表命名:**
```
推荐:
- 销售数据_2025
- 客户信息_华东区
- 月度汇总_12月

避免:
- Sheet1(无意义)
- 表1、表2(无意义)
- 特殊字符:* ? / \ [ ]
- 过长名称(>31字符)
```

**单元格/区域命名:**
```
使用场景:
- 常用数据区域
- 重要参数单元格
- 公式引用

命名方法:
1. 选中区域
2. 名称框输入名称(如:税率)
3. Enter确认

公式引用:
=销售额*税率
(替代=B2*$F$1)
```

### 4.3 数据清洗

**删除重复项:**
```
数据→删除重复项
- 选择关键列
- 保留第一条/最后一条
```

**分列:**
```
数据→分列

应用场景:
- 姓名分列:张三 → 张|三
- 日期分列:2025-12-21 → 2025|12|21

分隔符:
- 空格
- 逗号
- 制表符
- 自定义
```

**文本格式转换:**
```
去除前后空格:
=TRIM(A1)

统一大小写:
=UPPER(A1) 大写
=LOWER(A1) 小写
=PROPER(A1) 首字母大写

数字文本转数值:
=VALUE(A1)
或 A1*1
```

### 4.4 数据保护

**工作表保护:**
```
审阅→保护工作表

可设置:
- 允许选择锁定/未锁定单元格
- 允许格式化单元格
- 允许排序/筛选
- 密码保护

解除保护:
审阅→撤销工作表保护
```

**单元格锁定:**
```
默认情况:
- 所有单元格锁定状态:开启
- 但只有保护工作表后才生效

部分单元格可编辑:
1. 选中可编辑单元格
2. Ctrl+1→保护→取消勾选"锁定"
3. 保护工作表
```
