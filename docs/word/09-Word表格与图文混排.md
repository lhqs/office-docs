# Word表格与图文混排

## 本章导读

表格和图文混排是Word文档中最常用的排版元素,掌握这些技能能让文档更专业、美观、易读。本章将系统讲解Word表格的创建、美化和高级应用,以及图片、文本的混合排版技巧。

**本章核心内容:**
- 表格创建与快速操作
- 表格美化与格式设置
- 表格数据处理与计算
- 图片插入与格式调整
- 文字环绕与图文排版
- SmartArt图形应用

**适用场景:**
- 数据展示与对比
- 组织架构图制作
- 产品说明文档
- 图文并茂的报告
- 宣传单页设计

**学习目标:**
- 快速创建专业表格
- 掌握图文混排技巧
- 提升文档视觉效果
- 学会使用SmartArt

## 一、表格创建与基础操作

### 1.1 表格创建方法

**方法1: 鼠标拖动创建**
```
插入 → 表格 → 拖动鼠标选择行列数
优点:直观快速
适用:10×8以内的小表格
```

**方法2: 插入表格对话框**
```
插入 → 表格 → 插入表格
输入:列数、行数
自动调整:固定列宽/根据内容/根据窗口
优点:精确控制
适用:任意大小表格
```

**方法3: 绘制表格**
```
插入 → 表格 → 绘制表格
鼠标绘制表格边框和分隔线
优点:灵活自由,可创建不规则表格
适用:复杂表格、不规则合并
```

**方法4: 文本转表格**
```
步骤1: 准备文本数据(用Tab或逗号分隔)
姓名	年龄	部门
张三	28	技术部
李四	32	市场部

步骤2: 选中文本
步骤3: 插入 → 表格 → 文本转换成表格
步骤4: 选择分隔符:制表符/逗号/其他
步骤5: 确定,自动生成表格
```

**方法5: 快速表格**
```
插入 → 表格 → 快速表格
选择内置模板:
   - 简单表格
   - 带子标题表格
   - 列表表格
   - 日历表格
优点:一键套用样式
```

### 1.2 表格快捷操作

**快捷键:**

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 新建行 | `Tab`(最后一格) | 自动在表格末尾添加新行 |
| 换行 | `Enter` | 单元格内换行 |
| 制表符 | `Ctrl+Tab` | 单元格内插入Tab |
| 选中列 | 点击列顶端 | 鼠标变为向下箭头时点击 |
| 选中行 | 点击行左侧 | 鼠标变为向右箭头时点击 |
| 选中单元格 | 三击单元格 | 快速选中当前单元格 |
| 选中整个表格 | Alt+数字键盘5 | NumLock关闭状态 |

**鼠标操作:**
```
调整列宽:
   - 拖动列边线
   - 双击列边线:自动适应内容宽度

调整行高:
   - 拖动行边线
   - 双击行边线:自动适应内容高度

移动表格:
   - 拖动表格左上角的十字图标

调整表格大小:
   - 拖动表格右下角的方框
```

### 1.3 插入与删除

**插入行列:**
```
方法1: 右键菜单
   右键行/列 → 插入 → 选择插入位置

方法2: 布局选项卡
   表格工具-布局 → 插入
   - 在上方插入
   - 在下方插入
   - 在左侧插入
   - 在右侧插入

方法3: Tab键
   光标在最后一格按Tab:自动添加新行
```

**删除行列:**
```
选中要删除的行/列
右键 → 删除单元格/行/列
或:表格工具-布局 → 删除
```

**注意事项:**
```
⚠️ 删除与清空内容的区别:
   Delete键:清空内容,保留单元格
   删除单元格:移除单元格本身

⚠️ 删除整个表格:
   选中表格 → 右键 → 删除表格
   或:表格工具-布局 → 删除 → 删除表格
```

### 1.4 合并与拆分单元格

**合并单元格:**
```
步骤1: 选中要合并的单元格
步骤2: 右键 → 合并单元格
或:表格工具-布局 → 合并单元格

应用场景:
   - 表头合并
   - 数据分组标题
   - 跨列/跨行内容
```

**拆分单元格:**
```
步骤1: 选中要拆分的单元格
步骤2: 右键 → 拆分单元格
步骤3: 设置拆分的列数和行数
步骤4: 确定

应用场景:
   - 细化数据划分
   - 修正错误合并
```

**拆分表格:**
```
光标定位在要拆分的行
表格工具-布局 → 拆分表格(Ctrl+Shift+Enter)
表格一分为二

应用场景:
   - 在表格中间插入文字
   - 将大表格分为多个小表格
```

### 1.5 表格选择技巧

**精确选择:**

```
选择单元格:
   - 单击单元格内侧左边缘(黑色箭头)
   - 或三击单元格

选择一行:
   - 单击行左侧(空白处)
   - 鼠标变为向右箭头

选择一列:
   - 单击列顶端边线
   - 鼠标变为向下黑箭头

选择多行/多列:
   - 拖动选择
   - 或Shift+点击

选择整个表格:
   - 单击表格左上角十字图标
   - 或Alt+数字5(NumLock关闭)
   - 或四次点击表格
```

## 二、表格美化与格式设置

### 2.1 表格样式快速应用

**使用内置样式:**
```
步骤1: 选中表格
步骤2: 表格工具-设计 → 表格样式
步骤3: 选择喜欢的样式
   - 普通表格
   - 彩色表格
   - 网格表格
   - 列表表格

步骤4: 自定义选项
   ☑ 标题行:第一行特殊格式
   ☑ 汇总行:最后一行特殊格式
   ☑ 镶边行:奇偶行不同颜色
   ☑ 第一列:第一列特殊格式
   ☑ 最后一列:最后一列特殊格式
   ☑ 镶边列:奇偶列不同颜色
```

**修改表格样式:**
```
右键表格样式 → 修改表格样式
可自定义:
   - 边框
   - 底纹
   - 字体
   - 对齐方式
   - 应用范围(整个表格/标题行/数据行...)
```

### 2.2 边框与底纹

**设置边框:**
```
选中表格/单元格
表格工具-设计 → 边框 → 边框和底纹

边框设置:
   - 线型:实线/虚线/双线...
   - 宽度:0.5磅-6磅
   - 颜色:任意颜色
   - 应用于:单元格/表格

预设样式:
   - 无:无边框
   - 框:外框线
   - 全部:所有边框
   - 网格:内外边框
   - 自定义:点击预览区单独设置每条边
```

**设置底纹:**
```
表格工具-设计 → 底纹

选项:
   - 填充:单色填充
   - 图案样式:斜线/方格等
   - 图案颜色:图案颜色
   - 应用于:单元格/表格

常用配色:
   - 表头:深蓝色(#4472C4)
   - 数据行:浅灰色(#F2F2F2)交替
   - 汇总行:浅黄色(#FFF2CC)
```

**专业表格配色方案:**
```
方案1:商务蓝
   表头:RGB(68, 114, 196) - 深蓝
   数据:白色/RGB(217, 225, 242) - 浅蓝 交替

方案2:清新绿
   表头:RGB(112, 173, 71) - 深绿
   数据:白色/RGB(226, 239, 218) - 浅绿 交替

方案3:典雅灰
   表头:RGB(89, 89, 89) - 深灰
   数据:白色/RGB(242, 242, 242) - 浅灰 交替
```

### 2.3 对齐与文本方向

**对齐方式:**
```
表格工具-布局 → 对齐方式

9种对齐:
   顶端左对齐   顶端居中   顶端右对齐
   居中左对齐   居中      居中右对齐
   底端左对齐   底端居中   底端右对齐

快捷键:
   - Ctrl+E: 水平居中
   - Ctrl+L: 左对齐
   - Ctrl+R: 右对齐
```

**文本方向:**
```
表格工具-布局 → 文字方向

选项:
   - 水平(默认)
   - 垂直(从上到下)
   - 垂直(从下到上)

应用场景:
   - 表头竖排显示
   - 窄列标题
   - 特殊设计需求
```

**单元格边距:**
```
表格工具-布局 → 单元格边距

设置:
   - 上:0.05厘米
   - 下:0.05厘米
   - 左:0.19厘米
   - 右:0.19厘米

作用:
   文字与单元格边框的间距,增加美观性
```

### 2.4 行高与列宽

**精确设置行高:**
```
方法1:对话框设置
   表格工具-布局 → 高度
   输入具体数值(如0.8厘米)

方法2:表格属性
   右键表格 → 表格属性 → 行
   勾选"指定高度"
   输入数值
   选择度量单位:厘米/磅/行

方法3:标尺拖动
   拖动表格左侧标尺的行标记
```

**精确设置列宽:**
```
方法1:直接输入
   表格工具-布局 → 宽度
   输入数值

方法2:平均分布列宽
   选中需要调整的列
   表格工具-布局 → 分布列
   所有列宽度相等

方法3:自动调整
   表格工具-布局 → 自动调整
   - 根据内容自动调整
   - 根据窗口自动调整
   - 固定列宽
```

**平均分布行高:**
```
选中多行
表格工具-布局 → 分布行
所有行高度相等
```

### 2.5 表格定位与属性

**表格定位:**
```
右键表格 → 表格属性 → 表格

对齐方式:
   - 左对齐
   - 居中
   - 右对齐

文字环绕:
   - 无:表格独占一行
   - 环绕:文字环绕在表格周围

定位(当选择环绕时):
   - 水平位置:相对于列/页边距/页面
   - 垂直位置:相对于段落/页边距/页面
   - 与文字的距离:上下左右间距
```

**表格选项:**
```
表格属性 → 选项

设置:
   - 默认单元格边距
   - 默认单元格间距(制作特殊效果)
   - 自动重调尺寸以适应内容
```

## 三、表格数据处理

### 3.1 排序

**数据排序:**
```
步骤1: 选中要排序的数据(不含表头)
步骤2: 表格工具-布局 → 排序
步骤3: 设置排序依据
   主要关键字:选择列(如"成绩")
   类型:数字/文本/日期
   升序/降序

步骤4: 添加次要关键字(可选)
   当主关键字相同时的排序规则

步骤5: 勾选"有标题行"(如果第一行是表头)
步骤6: 确定
```

**排序示例:**
```
学生成绩表排序:
   主要:按成绩降序(高到低)
   次要:按姓名升序(A-Z)

日期表排序:
   类型选择"日期"
   自动识别日期格式排序
```

### 3.2 表格计算

**插入公式:**
```
光标定位在要显示结果的单元格
表格工具-布局 → 公式

常用函数:
   =SUM(LEFT):求左侧所有单元格的和
   =SUM(RIGHT):求右侧单元格的和
   =SUM(ABOVE):求上方单元格的和
   =SUM(BELOW):求下方单元格的和
   =AVERAGE(LEFT):求平均值
   =MAX(LEFT):求最大值
   =MIN(LEFT):求最小值
   =COUNT(LEFT):计数
   =PRODUCT(LEFT):求乘积
```

**单元格引用:**
```
类似Excel的引用方式:
   A1:第1列第1行
   B2:第2列第2行

公式示例:
   =SUM(A1:A5):求A1到A5的和
   =A1*B1:两个单元格相乘
   =SUM(A1:D1):求一行的和
```

**更新域:**
```
公式是域代码,数据改变后需要更新:
   右键公式单元格 → 更新域
   或:选中公式 → F9

批量更新:
   Ctrl+A 全选表格 → F9
```

**注意事项:**
```
⚠️ Word表格计算功能有限
   适用:简单的求和、平均值
   不适用:复杂的数据分析

⚠️ 复杂运算建议使用Excel
   可在Word中嵌入Excel表格
   插入 → 对象 → Microsoft Excel工作表
```

### 3.3 表格与Excel互转

**Excel表格粘贴到Word:**
```
方法1:保留源格式
   Excel复制 → Word粘贴(Ctrl+V)
   粘贴选项:保留源格式
   结果:保留Excel格式,但失去计算功能

方法2:链接Excel
   Excel复制 → Word → 选择性粘贴
   选择:粘贴链接
   选择:Microsoft Excel工作表对象
   结果:Excel数据更新,Word自动更新

方法3:嵌入Excel
   插入 → 对象 → Microsoft Excel工作表
   结果:可在Word中直接编辑Excel表格
```

**Word表格转Excel:**
```
方法1:复制粘贴
   Word中复制表格 → Excel中粘贴
   自动转换为Excel表格

方法2:另存为网页
   文件 → 另存为 → 网页
   用Excel打开html文件
   另存为xlsx格式
```

### 3.4 重复标题行

**设置重复标题行:**
```
应用场景:
   跨页表格,每页都显示表头

操作步骤:
   选中表头行(可以是多行)
   表格工具-布局 → 重复标题行
   或:右键 → 表格属性 → 行 → 勾选"在各页顶端以标题行形式重复出现"

效果:
   表格跨页时,每页顶端自动重复表头
```

**注意事项:**
```
✓ 只能重复表格开头的连续行
✓ 重复的标题行不能编辑(在后续页)
✓ 要修改需回到第一页修改原标题行
```

## 四、图片插入与处理

### 4.1 插入图片

**插入本地图片:**
```
插入 → 图片 → 此设备
浏览并选择图片文件
支持格式:jpg, png, gif, bmp, tiff等
```

**插入在线图片:**
```
插入 → 图片 → 联机图片
搜索必应图片搜索结果
选择并插入

⚠️ 注意版权问题
   商用文档请使用正版图片
```

**插入屏幕截图:**
```
插入 → 屏幕截图
   - 可用窗口:选择打开的窗口截图
   - 屏幕剪辑:自由截取屏幕区域(类似微信截图)

快捷方式:
   Windows: Win+Shift+S(Win10/11系统自带截图)
   截图后Ctrl+V粘贴到Word
```

**插入形状和图标:**
```
插入 → 形状:矩形、圆形、箭头、流程图等
插入 → 图标:Office自带的矢量图标(支持换色)
插入 → 3D模型:插入三维模型(Office 365)
```

### 4.2 图片格式调整

**调整大小:**
```
方法1:拖动控制点
   按住Shift拖动角控制点:等比例缩放
   直接拖动边控制点:任意缩放

方法2:精确输入
   图片工具-格式 → 大小
   输入高度和宽度数值
   勾选"锁定纵横比":等比例缩放

方法3:布局选项
   右键图片 → 大小和位置
   精确设置尺寸
```

**裁剪图片:**
```
选中图片
图片工具-格式 → 裁剪

裁剪选项:
   - 裁剪:拖动裁剪框
   - 裁剪为形状:圆形、星形等
   - 纵横比:1:1(正方形)、16:9、4:3等

操作技巧:
   - Ctrl+拖动:对称裁剪
   - Shift+拖动:等比例裁剪
   - 裁剪后再次点击裁剪可恢复
```

**压缩图片:**
```
选中图片
图片工具-格式 → 压缩图片

选项:
   ☑ 应用于:此图片/文档中所有图片
   ☑ 删除图片的裁剪区域

分辨率:
   - 高保真度:330 PPI(最佳质量)
   - 打印:220 PPI(印刷质量)
   - Web:150 PPI(网页使用)
   - 电子邮件:96 PPI(屏幕查看)

建议:
   普通文档选择150-220 PPI平衡质量和文件大小
```

### 4.3 图片样式与效果

**应用图片样式:**
```
选中图片
图片工具-格式 → 图片样式

内置样式:
   - 简单边框
   - 柔化边缘
   - 阴影效果
   - 倒影效果
   - 发光效果
   - 三维旋转
```

**图片效果:**
```
图片工具-格式 → 图片效果

效果类型:
   - 阴影:外阴影、内阴影
   - 映像:倒影效果
   - 发光:外发光、内发光
   - 柔化边缘:模糊边缘
   - 棱台:3D立体感
   - 三维旋转:透视效果
```

**图片边框:**
```
图片工具-格式 → 图片边框
   - 颜色:选择边框颜色
   - 粗细:0.25磅-6磅
   - 虚线:实线/虚线/点线等
```

**艺术效果:**
```
图片工具-格式 → 艺术效果
   - 水彩海绵
   - 铅笔素描
   - 粉笔素描
   - 马赛克泡泡
   - 毛玻璃
   - ...
```

### 4.4 图片颜色调整

**更正:**
```
图片工具-格式 → 更正

亮度和对比度:
   - 提高亮度
   - 降低亮度
   - 增加对比度
   - 降低对比度

锐化和柔化:
   - 锐化:使图片更清晰
   - 柔化:使图片更柔和
```

**颜色:**
```
图片工具-格式 → 颜色

饱和度:
   - 饱和度0%:黑白效果
   - 饱和度100%:原始颜色
   - 饱和度400%:过饱和效果

色调:
   - 温度11200K:偏暖色调
   - 温度4700K:偏冷色调

重新着色:
   - 灰度:黑白照片
   - 深褐色:怀旧效果
   - 单色:指定主题色
```

**设置透明色:**
```
图片工具-格式 → 颜色 → 设置透明色
点击图片中要变透明的颜色
该颜色变为透明

适用:
   简单纯色背景的图片去背景
```

**删除背景:**
```
选中图片
```

## 五、文字环绕与图文排版

### 5.1 文字环绕方式

**环绕方式类型:**

```
右键图片 → 环绕文字 → 选择方式
或:图片工具-格式 → 环绕文字
```

| 环绕方式 | 效果 | 适用场景 |
|----------|------|----------|
| 嵌入型 | 图片作为文字,随文字移动 | 默认方式,图片小,与文字在一行 |
| 四周型 | 文字环绕图片四周(矩形框) | 最常用,图片规则 |
| 紧密型 | 文字紧密环绕图片轮廓 | 图片不规则形状 |
| 穿越型 | 文字穿过图片内部 | 特殊效果,图片有透明区域 |
| 上下型 | 文字在图片上下,左右留空 | 图片独占一行 |
| 衬于文字下方 | 图片作为背景 | 水印、背景图 |
| 浮于文字上方 | 图片覆盖文字 | 标注、强调 |

**常用组合:**
```
正文插图:四周型/紧密型
   文字自然环绕图片

图片水印:衬于文字下方
   降低图片透明度
   图片工具-格式 → 透明度:50%

图片标注:浮于文字上方
   可自由拖动位置
```

### 5.2 图片定位

**版式设置:**
```
右键图片 → 大小和位置 → 位置

水平对齐:
   - 对齐方式:左/居中/右
   - 相对于:页边距/页面/栏

垂直对齐:
   - 对齐方式:顶端/居中/底端
   - 相对于:页边距/页面/段落

绝对位置:
   - 水平:右侧:2厘米 相对于:页边距
   - 垂直:下侧:5厘米 相对于:页边距
```

**锁定图片位置:**
```
大小和位置 → 位置

选项:
   ☑ 对象随文字移动:图片随段落移动
   ☐ 对象随文字移动:图片固定在页面上
   ☑ 锁定标记:图片不能拖动
```

**常用定位:**
```
右上角Logo:
   水平:右对齐 相对于页边距
   垂直:顶端对齐 相对于页边距
   取消"随文字移动"

正文插图:
   环绕方式:四周型
   勾选"随文字移动"
   图片跟随段落位置
```

```

作用:
```

**旋转与翻转:**

```
   - 其他旋转选项:精确输入角度
```

```
   1. 插入 → 形状 → 流程图形状
      - 开始/结束:圆角矩形/椭圆
      - 过程:矩形
      - 判断:菱形
      - 数据:平行四边形

   2. 插入 → 形状 → 线条 → 箭头连接符
      连接各形状

   3. 统一样式
      选中所有形状
      形状工具-格式 → 统一设置

   4. 对齐分布
      选择窗格管理对象
      对齐工具整理布局
```

## 十、高频问题Q&A

**Q1: 表格跨页时如何让每页都显示表头?**
```
A: 选中表头行 → 表格工具-布局 → 重复标题行
   或:表格属性 → 行 → 勾选"在各页顶端以标题行形式重复出现"
```

**Q2: 如何让表格某一行不跨页断开?**
```
A: 选中该行 → 右键 → 表格属性 → 行
   取消勾选"允许跨页断行"
```

**Q3: 图片插入后显示不全怎么办?**
```
A: 可能是行距固定值导致
   解决:选中图片所在段落 → 段落设置
   行距改为"单倍行距"或"多倍行距"
```

**Q4: 如何去除表格双线边框?**
```
A: 表格工具-设计 → 边框 → 单线边框
   或:边框和底纹 → 选择单线线型
```

**Q5: Word图片如何保存为独立文件?**
```
A: 方法1:右键图片 → 另存为图片
   方法2:文件 → 另存为 → 网页
   在保存的文件夹中找到图片文件
```

**Q6: SmartArt能否转换为形状单独编辑?**
```
A: 可以
   选中SmartArt → SmartArt工具-设计 → 转换 → 转换为形状
   转换后失去SmartArt特性,但可单独编辑每个形状
```

**Q7: 如何制作无边框表格?**
```
A: 表格工具-设计 → 边框 → 无框线
   或:用表格布局,设置边框为白色或无边框
```

**Q8: 图片如何设置为背景平铺?**
```
A: 设计 → 页面颜色 → 填充效果 → 图片
   选择图片,不勾选"锁定图片纵横比"
```

**Q9: 表格文字垂直居中对齐?**
```
A: 选中单元格 → 表格工具-布局 → 对齐方式
   选择居中或其他垂直+水平组合对齐
```

**Q10: 如何快速给图片编号?**
```
A: 右键图片 → 插入题注
   标签选择"图"
   自动编号:图1, 图2, 图3...
```
