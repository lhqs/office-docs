# 邮件自动化处理

## 本章概览

邮件自动化能显著提升处理效率。本章介绍快速步骤、规则设置、模板应用等自动化技巧。

**学习目标**
- 掌握快速步骤使用
- 学会创建高级规则
- 熟练应用邮件模板
- 了解VBA自动化

---

## 45.1 快速步骤

### 45.1.1 内置快速步骤

**常用快速步骤**
```
主页 → 快速步骤

默认快速步骤:
- 移至: 移动到文件夹
- 团队电子邮件: 转发给团队
- 完成: 标记完成并移至文件夹

使用:
选中邮件 → 点击快速步骤
或快捷键Ctrl + Shift + 1-9
```

### 45.1.2 自定义快速步骤

**创建步骤**
```
快速步骤 → 新建 → 自定义

示例1: 快速归档
名称: 归档已读
操作:
1. 标记为已读
2. 移动到"存档"文件夹
3. 分类为"已处理"(绿色)

示例2: 转发并标记
名称: 转发给经理
操作:
1. 转发到: manager@company.com
2. 标记跟进(明天)
3. 移动到"等待回复"文件夹

示例3: 快速回复
名称: 已收到回复
操作:
1. 回复
2. 插入文字: "已收到,正在处理,稍后回复详细信息。"
3. 发送
```

**高级设置**
```
为步骤分配快捷键:
Ctrl + Shift + 1-9

工具提示:
显示说明文字

选项:
☑ 使用后显示选项
□ 完成后关闭窗口
```

## 45.2 规则自动化

### 45.2.1 基础规则

**创建规则**
```
文件 → 管理规则和通知 → 新建规则

步骤1: 选择条件
- 发件人为特定人员
- 主题包含特定字词
- 抄送给我
- 仅发送给我
- 标记为重要
- 附件大小

步骤2: 选择操作
- 移动到文件夹
- 分类
- 标记
- 转发
- 播放声音
- 删除

步骤3: 设置例外(可选)
- 除非...

步骤4: 命名并启用
```

**实用规则示例**
```
规则1: 自动分类项目邮件
条件: 主题包含"[项目A]"
操作: 移动到"项目A"文件夹 + 分类(红色)

规则2: 重要邮件提醒
条件: 发件人为"boss@company.com"
操作: 播放声音 + 标记为重要 + 显示桌面通知

规则3: 自动转发
条件: 抄送给我 且 主题包含"紧急"
操作: 转发给"assistant@company.com"

规则4: 垃圾邮件过滤
条件: 主题包含"广告"或"优惠"
操作: 移动到"垃圾邮件"文件夹

规则5: 自动归档
条件: 发送日期超过30天 且 已读
操作: 移动到"2024年归档"文件夹
```

### 45.2.2 高级规则

**组合条件**
```
多条件AND:
条件1: 发件人为"client@company.com"
条件2: 主题包含"合同"
操作: 移动到"重要合同"+ 标记高优先级

多条件OR:
使用多个规则实现
```

**复杂操作**
```
多操作组合:
1. 移动到文件夹
2. 分类
3. 标记跟进
4. 转发副本
5. 标记为已读

顺序执行:
按规则列表从上到下执行
可调整优先级
```

### 45.2.3 规则管理

**规则优先级**
```
管理规则和通知 → 调整顺序

技巧:
- 具体规则在前
- 通用规则在后
- 删除操作规则最后

示例顺序:
1. 老板邮件(最高优先级)
2. 紧急项目邮件
3. 客户邮件
4. 一般项目邮件
5. 新闻订阅
6. 垃圾邮件
```

**规则测试**
```
运行规则:
文件 → 管理规则和通知 → 立即运行规则

选择:
- 规则名称
- 应用文件夹
- 应用于所有邮件/未读邮件

测试:
在"已发送邮件"文件夹测试
避免影响收件箱
```

```
模板2: 周报

```

```
2. 选中文字
```

### 45.5.2 简单脚本

**自动回复脚本**
```vba
Sub AutoReply()
    Dim objItem As MailItem
    Dim objReply As MailItem

    Set objItem = Application.ActiveExplorer.Selection.Item(1)
    Set objReply = objItem.Reply

    With objReply
        .Body = "您的邮件已收到,我会尽快回复。" & vbCrLf & vbCrLf & .Body
        .Send
    End With
End Sub
```

**批量转发脚本**
```vba
Sub ForwardToTeam()
    Dim objItem As MailItem
    Dim objFwd As MailItem

    For Each objItem In Application.ActiveExplorer.Selection
        Set objFwd = objItem.Forward
        objFwd.Recipients.Add "team@company.com"
        objFwd.Send
    Next
End Sub
```

**定时发送脚本**
```vba
Sub DelayedSend()
    Dim objMail As MailItem
    Set objMail = Application.CreateItem(olMailItem)

    With objMail
        .To = "recipient@company.com"
        .Subject = "定时邮件"
        .Body = "这是一封定时发送的邮件"
        .DeferredDeliveryTime = DateAdd("h", 2, Now) '2小时后发送
        .Send
    End With
End Sub
```

## 45.7 实战案例

### 案例1: 客户邮件自动化

**场景**
```
每天收到大量客户邮件
需要快速响应和分类
```

**自动化方案**
```
1. 创建规则:
   规则1: VIP客户
   条件: 发件人在VIP列表
   操作: 标记重要+播放声音+移至VIP文件夹

   规则2: 新客户
   条件: 发件人不在联系人
   操作: 移至"新客户"文件夹

   规则3: 咨询类
   条件: 主题包含"咨询"
   操作: 移至"咨询"文件夹

2. 快速步骤:
   "快速回复":
   - 回复并插入: "感谢咨询,我们会在24小时内回复"
   - 标记跟进(明天)

   "转销售":
   - 转发给sales@company.com
   - 标记完成

3. 模板:
   - 产品介绍模板
   - 报价单模板
   - 感谢信模板

效果:
邮件处理时间减少60%
响应速度提升3倍
```

### 案例2: 项目邮件管理

**自动化流程**
```
1. 命名规范:
   要求团队发邮件使用前缀
   [项目A] [项目B] [项目C]

2. 规则自动分类:
   规则: 主题包含"[项目A]"
   操作: 移至"项目A"文件夹+分类红色

3. 搜索文件夹:
   "项目紧急事项"
   条件: 包含"[项目]"且标记为重要

4. VBA脚本:
   每周五自动生成项目邮件汇总
   发送给项目经理

5. 快速步骤:
   "项目完成归档"
   - 标记完成
   - 移至"已完成项目"
   - 取消跟进标记
```
