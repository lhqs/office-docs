# 日历与会议管理

## 本章概览

Outlook日历是强大的时间和会议管理工具。本章介绍如何使用日历高效管理日程、组织会议。

**学习目标**
- 掌握日历基本操作
- 学会会议组织流程
- 了解共享日历使用
- 提升时间管理能力

---

## 43.1 日历基础

### 43.1.1 日历视图

**切换日历**
```
快捷键: Ctrl + 2

视图模式:
- 日视图: 查看单日详细日程
- 工作周: 周一至周五
- 周视图: 完整一周
- 月视图: 月度概览

切换:
主页 → 排列 → 选择视图
或点击底部视图切换按钮
```

**时间设置**
```
文件 → 选项 → 日历

工作时间:
开始: 09:00
结束: 18:00
工作日: 周一至周五

时区设置:
主时区: (GMT+08:00)北京
显示第二时区(跨时区协作)

第一周:
星期日/星期一开始
```

### 43.1.2 创建约会

**新建约会**
```
快捷键: Ctrl + N(日历视图中)

基本信息:
主题: 约会名称
位置: 地点
开始时间: 2024-01-20 14:00
结束时间: 2024-01-20 15:00

选项:
☑ 显示为: 忙碌/空闲/暂定/外出
☑ 提醒: 提前15分钟
□ 全天事件
□ 定期约会
```

**全天事件**
```
勾选"全天事件"

用途:
- 节假日
- 生日
- 截止日期
- 出差行程

技巧:
设置为"空闲"
不占用工作时间
```

### 43.1.3 定期约会

**重复模式**
```
约会 → 定期约会

模式:
- 每日: 工作日/每天
- 每周: 每周一/周三/周五
- 每月: 每月15日/第一个周一
- 每年: 每年1月1日

范围:
开始日期: 2024-01-01
结束: 无结束日期/结束于XX/出现X次

示例:
每周一上午10:00团队例会
每月最后一个工作日月度总结
```

## 43.2 会议管理

### 43.2.1 创建会议

**会议vs约会**
```
约会: 个人日程,不涉及他人
会议: 邀请他人参加的日程

创建会议:
主页 → 新建会议
或约会中点击"邀请与会者"
```

**会议邀请**
```
必填项:
收件人: 参会人员邮箱
主题: 会议主题
时间: 开始和结束时间
位置: 会议室或在线链接

可选项:
正文: 会议议程、准备事项
附件: 会议资料
提醒: 提前通知时间

发送:
点击"发送"发出邀请
```

### 43.2.2 会议响应

**接受/拒绝会议**
```
收到邮件后:
- 接受: 加入日历
- 暂定: 待定(可能参加)
- 拒绝: 不参加
- 建议新时间: 时间冲突时

响应选项:
☑ 编辑响应(说明理由)
☑ 发送响应
□ 不发送响应
```

**查看回复**
```
组织者视图:
打开会议 → 跟踪

显示:
✓ 已接受: 5人
? 暂定: 2人
✗ 已拒绝: 1人
○ 无响应: 3人

操作:
提醒未响应者
调整参会人员
```

### 43.2.3 会议议程

**议程模板**
```
会议主题: XX项目进度会
时间: 2024-01-20 14:00-15:00
地点: 3楼会议室A

议程:
1. 项目进展汇报(20分钟)
2. 问题讨论(30分钟)
3. 下一步计划(10分钟)

参会人员:
必须: 项目经理、技术负责人
可选: 相关开发人员

会前准备:
请准备项目进度报告
```

## 43.3 日程安排技巧

### 43.3.1 时间分类

**颜色标注**
```
右键约会 → 分类

颜色分类:
红色: 重要会议
橙色: 项目会议
黄色: 待定事项
绿色: 个人时间
蓝色: 培训学习
紫色: 出差行程

快捷键:
Ctrl + F2-F12: 快速分类
```

### 43.3.2 时间块管理

**时间块划分**
```
上午:
09:00-10:00 重要工作时间(深度工作)
10:00-12:00 会议时间

下午:
14:00-16:00 项目执行时间
16:00-17:00 邮件处理、沟通
17:00-18:00 计划和总结

技巧:
- 预留专注工作时间
- 集中安排会议
- 留出缓冲时间
```

**日程模板**
```
创建模板:
周一: 团队例会
周三: 项目会议
周五: 周总结

批量添加:
复制约会
粘贴到其他日期
调整具体时间
```

### 43.3.3 避免时间冲突

**空闲/忙碌查询**
```
创建会议时:
日程安排助理 → 查看所有与会者状态

显示:
- 绿色: 空闲
- 蓝色: 忙碌
- 紫色: 外出
- 灰色: 无信息

自动建议:
查找可用时间(所有人都空闲)
```

## 43.4 共享日历

### 43.4.1 共享设置

**共享日历**
```
主页 → 共享日历

共享对象:
输入邮箱地址

权限级别:
- 可以查看忙/闲时间
- 可以查看标题和位置
- 可以查看所有详细信息
- 编辑者
- 代理

发送邀请
```

**接受共享**
```
收到共享邀请:
点击"接受"

查看:
在"共享日历"节点下
显示他人日历

技巧:
并排查看多个日历
叠加显示
```

### 43.4.2 团队日历

**创建团队日历**
```
用途:
查看团队成员日程
协调会议时间
了解出差/休假

设置:
创建Office 365群组日历
或在SharePoint创建日历
团队成员共享访问

应用:
- 部门日历
- 项目日历
- 公司假期日历
```

### 43.4.3 资源日历

**会议室预订**
```
组织设置会议室资源:
Exchange管理员添加会议室

预订:
1. 创建会议
2. 添加会议室到"位置"或"收件人"
3. 查看会议室空闲时间
4. 发送邀请

自动接受:
会议室设置自动接受预订
避免冲突自动拒绝
```

## 43.5 提醒与通知

### 43.5.1 提醒设置

**默认提醒**
```
文件 → 选项 → 日历 → 默认提醒

约会和会议默认提醒:
☑ 默认提醒: 15分钟

全天事件提醒:
18小时(前一天晚上)

修改单个约会:
打开约会 → 提醒 → 选择时间
```

**提醒时间**
```
常用提醒:
- 0分钟(开始时)
- 5分钟
- 15分钟
- 30分钟
- 1小时
- 1天
- 自定义

技巧:
重要会议设置多个提醒
```

### 43.5.2 提醒管理

**处理提醒**
```
提醒弹窗:
- 消除: 关闭提醒
- 推迟: 稍后再提醒(5分钟/15分钟等)
- 打开项目: 查看详情

全部消除:
一次关闭所有提醒

技巧:
推迟功能避免遗漏
重要事项不要总推迟
```

## 43.8 实战案例

### 案例1: 跨部门会议组织

**场景**
```
组织跨部门项目启动会
涉及5个部门,20人
需要会议室和投影设备
```

**操作流程**
```
1. 确定时间:
   查看关键人员空闲时间
   使用日程安排助理
   找到所有人都空闲的时段

2. 预订资源:
   添加会议室到邀请
   添加投影设备资源

3. 发送邀请:
   收件人: 必须参加人员
   可选: 备选人员
   正文: 会议议程和准备事项
   附件: 项目资料

4. 跟踪响应:
   查看回复情况
   提醒未响应者
   确认最终参会名单

5. 会前提醒:
   前一天发送提醒邮件
   确认会议室设备
```

### 案例2: 个人时间管理

**每日日程**
```
07:00-08:00 晨间锻炼(个人)
09:00-10:00 深度工作时间(预留)
10:00-11:00 处理邮件和沟通
11:00-12:00 项目工作
14:00-15:00 团队会议
15:00-17:00 项目执行
17:00-18:00 邮件、总结、计划

技巧:
- 设置定期约会
- 颜色区分类型
- 预留缓冲时间
- 每周回顾调整
```

## 本章小结

**核心要点**
1. **日历视图**: 日/周/月视图切换
2. **约会管理**: 创建约会、定期约会
3. **会议组织**: 邀请、响应、跟踪
4. **共享日历**: 权限设置、团队协作
5. **提醒通知**: 设置提醒、处理提醒

**高效使用**
- 统一使用日历管理时间
- 颜色分类不同类型事务
- 共享日历便于协作
- 时间块管理提升效率
- 移动端随时查看

**最佳实践**
- 所有约会都记录在日历
- 会议前发送议程
- 及时响应会议邀请
- 定期回顾调整日程
- 预留专注工作时间
