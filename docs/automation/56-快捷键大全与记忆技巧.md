# 快捷键大全与记忆技巧

## 引言

快捷键是提升办公效率的最直接方法。熟练使用快捷键,可以让你的工作效率提升50%以上。本章将系统梳理Office三件套的常用快捷键,并分享记忆技巧。

## 五、快捷键记忆技巧

### 5.1 字母助记法

**Ctrl + 首字母:**
```
Ctrl+B → Bold(加粗)
Ctrl+C → Copy(复制)
Ctrl+F → Find(查找)
Ctrl+H → rePlace中的H(替换)
Ctrl+I → Italic(斜体)
Ctrl+K → linK(链接)
Ctrl+N → New(新建)
Ctrl+O → Open(打开)
Ctrl+P → Print(打印)
Ctrl+S → Save(保存)
Ctrl+U → Underline(下划线)
Ctrl+V → 粘贴(像一个向下的箭头)
Ctrl+X → 剪切(像剪刀交叉)
Ctrl+Z → 撤销(Z是字母表最后,回到过去)
```

| F7 | 拼写检查 | - |
| F8 | 扩展选择 | - |
| F9 | 计算公式 | - |
| F10 | 激活菜单栏 | - |
| F11 | 图表/全屏 | - |
| F12 | 另存为 | - |

### 5.3 组合记忆

**Shift组合:**
```
Shift = 扩展、反向
- Shift+方向键:扩展选择
- Shift+F5:反向(上一个位置)
- Shift+F3:反向(插入函数)
- Shift+Enter:反向换行(不分段)
```

**Alt组合:**
```
Alt = 激活菜单/快速访问
- Alt+F4:关闭程序
- Alt+Tab:切换程序
- Alt+Enter:单元格内换行
- Alt+N:插入选项卡
```

**Ctrl+Shift组合:**
```
双修饰键 = 强化版功能
- Ctrl+Shift+>:增大字号
- Ctrl+Shift+L:筛选
- Ctrl+Shift+Enter:数组公式
- Ctrl+Shift+方向键:扩展选择到边缘
```
